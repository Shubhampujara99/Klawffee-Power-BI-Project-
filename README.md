# Klawffee Sales Performance Infographic Dashboard

An interactive Power BI infographic designed to analyze retail sales performance, customer purchasing behavior, and revenue trends across multiple Klawffee store locations.

## Purpose

The Klawffee Sales Dashboard is a Power BI report built to analyze 149,116 sales transactions across three store locations. It focuses on sales trends, product performance, time-based purchasing behavior, and revenue forecasting to support data-driven decision-making.

## Tech Stack

The dashboard was built using the following tools and technologies:
- Power BI Desktop – Data modeling, visualization, and reporting
- DAX (Data Analysis Expressions) – KPI calculations, rankings, forecasting, and dynamic narratives
- Data Modeling – Snowflake schema with fact and dimension tables
- PowerPoint – Custom background design for the infographic layout
- File Formats – .pbix for development and .pdf for final presentation

## Data Source <br>

- Data Origin: Combination of GPT-generated synthetic data and Kaggle datasets
- Dataset Size: 149,116 rows and 11 columns

Structure: <br>
- Sales fact table with transaction-level data
- Dimension tables for products, stores, and dates
- Data was pre-cleaned; all modeling, analysis, and visualization steps were performed in Power BI.

## Features / Highlights
   
Business Problem: <br>
Retail teams often lack an intuitive way to analyze product performance, customer buying patterns, and store-level trends using raw transactional data.

Goals of the Dashboard: <br>
- To deliver an infographic-style analytical dashboard that:
- Tracks sales performance and customer behavior
- Highlights high- and low-performing products and time periods
- Supports revenue planning using forecasting and trend analysis

## Walkthrough of Key Visuals
### Key Performance Indicators: <br>
- Total Sales: 6,98,812
- Total Transactions: 149,116
- Average Order Size: 4.69
- Number of Stores: 3
- Number of Products: 80

### Product Performance Analysis: <br>
- Bar and column charts highlighting top-selling products and categories
- Ranking visuals to identify high and low performers

### Time-Based Sales Insights: <br>
- Line and column charts analyzing sales by hour, day, and month
- Matrix visuals using conditional formatting with value masking to emphasize patterns over raw numbers

### Forecasting and Targets: <br>
- Month-wise sales forecasts for each store
- Full-year revenue goal of 14,09,207 with comparative growth analysis

### Smart Narrative Storytelling: <br>
- Dynamic narratives explaining trends, store performance differences, and improvement opportunities
- Business Impact and Insights
- Identified peak sales hours and low-activity periods to support operational planning
- Enabled store-level revenue tracking against annual targets
- Highlighted top revenue-driving products and underperforming categories
- Delivered executive-friendly insights through a single-page infographic design
